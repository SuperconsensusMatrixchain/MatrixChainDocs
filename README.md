# Introduction

矩链使用说明文档

### web构建

注意：需要安装docker

1. 克隆目录

```
git clone https://github.com/superconsensus/MatrixChainDocs.git
```

2. make使用

```
cd MatrixChainDocs
// makefile 提供了3个命令 build / run /stop
// 首次运行需要构建镜像
make build
// 运行
make run
// 停止
make stop
```

3. 访问

访问本机的4000端口可以看到文档构建成功效果。
